# Diamond Air Chatbot

An AI-powered customer support chatbot for Diamond Air airlines, built with the Anthropic Claude API.

## Features
- Book flights conversationally
- Multi-city airport selection
- One-way and round-trip booking
- Payment processing simulation
- Escalate to human agent

## Setup
1. Get your Anthropic API key from [console.anthropic.com](https://console.anthropic.com)
2. Open `index.html` and replace `YOUR_API_KEY_HERE` with your key
3. Open `index.html` in a browser

## Built with
- HTML, CSS, JavaScript
- Anthropic Claude API (claude-sonnet-4)
